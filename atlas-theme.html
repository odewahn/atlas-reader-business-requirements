Feature Description

The Atlas HTML theme is where we will implement the key user facing features.  These include:

  • Visual identity that is consistent with the O'Reilly master brand strategy and the ongoing reconceptualization of oreilly.com
  • Ability to change elements in the CSS (for example, the color of the nave bar) or JavaScript (for example, the name of the Disqus forum) by placing variables in atlas.json
